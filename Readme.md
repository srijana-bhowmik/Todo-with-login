# Login with Todo App

## Description
This is a **responsive login system with a todo list**, built using **HTML, CSS, and JavaScript**.  
Users can:
- Register an account
- Login securely
- Create, view, and delete tasks in a personal todo list  

The interface is **responsive**, with the login form centered on desktop and stacked layout on mobile devices.  

---

## Features
- **Responsive Design:** Works on both desktop and mobile.
- **User Registration and Login:** Stores users in browser `localStorage`.
- **Todo List Management:** Add and delete tasks, tasks are saved per user.
- **Simple and Clean UI:** Rounded buttons, shadows, and image support.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser LocalStorage for data persistence
